Plugin prefix_with
In: /home/david/repos/complete/modules/common/plugins/puppet/parser/functions/prefix_with.rb

Prefixes arguments 2..n with first argument. 

 prefix_with(string prefix, string[] arguments) : string[]

Example: 

 prefix_with("php-", [ "blah", "foo" ])

will result in this array: 

 [ "php-blah", "php-foo" ]

Type

function

[Validate]
